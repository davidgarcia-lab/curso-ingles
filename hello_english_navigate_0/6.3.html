Sigamos con la gramática.

Each y All the

Each – cada, se usa para hablar de una cosa individual dentro de un grupo.

  • Each flat has got a kitchen. – Cada apartamento tiene cocina.
  • Each bathroom has got a shower. – Cada baño tiene ducha.
  • Each bedroom has got a bed. – Cada dormitorio tiene una cama.

All the – todos los o todas las, se usa para hablar del grupo completo.

  • All the flats have got internet. – Todos los apartamentos tienen internet.
  • All the kitchens have got a fridge. – Todas las cocinas tienen refrigerador.
  • All the balconies have got a table. – Todos los balcones tienen una mesa.

Atención a has got – tiene, con una sola cosa: each flat, each bedroom, each bathroom.

  • Each flat has got a TV. – Cada apartamento tiene televisor.
  • Each kitchen has got a microwave. – Cada cocina tiene microondas.

Y have got – tienen, con varias cosas: all the flats, all the rooms, all the kitchens.

  • All the flats have got a balcony. – Todos los apartamentos tienen balcón.
  • All the rooms have got a bed. – Todas las habitaciones tienen una cama.

Preguntas

Para preguntar, decimos:

  • Has each flat got a kitchen? – ¿Cada apartamento tiene cocina?
  • Has each bathroom got a shower? – ¿Cada baño tiene ducha?
  • Has each bedroom got a TV? – ¿Cada dormitorio tiene televisor?

Para el grupo completo, decimos:

  • Have all the flats got internet? – ¿Todos los apartamentos tienen internet?
  • Have all the rooms got a bed? – ¿Todas las habitaciones tienen cama?
  • Have all the kitchens got a microwave? – ¿Todas las cocinas tienen microondas?

Practiquemos pronunciación. En each /iːtʃ/, la vocal es larga: /iː/.

  • Each – cada
  • Each flat – cada apartamento
  • Each room – cada habitación

En all /ɔːl/, la vocal también es larga:

  • all the flats – todos los apartamentos

También practicamos linking – unión de sonidos. Cuando una palabra termina en consonante y la siguiente empieza en vocal, unimos suavemente. Has each se escucha como /hæz iːtʃ/.

  • Has each flat got a kitchen? – ¿Cada apartamento tiene cocina?

Otro ejemplo: got internet se une: /ɡɒt ˈɪntərnet/.

  • They have got internet. – Ellos tienen internet.

Y table and chairs se une: table and chairs – mesa y sillas.
